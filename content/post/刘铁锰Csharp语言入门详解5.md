## 操作符概览

| 类别               | 运算符                                                       |
| :----------------- | ------------------------------------------------------------ |
| 基本               | `x.y f(x) a[x] x++ x-- new typeof default checked unchecked delegate sizeof ->` |
| 一元               | `+ - ! ~ ++x --x (T)x await &x *x`                           |
| 乘法               | `* / %`                                                      |
| 加减               | `+ -`                                                        |
| 移位               | `<< >>`                                                      |
| 关系和类型检测     | `< > <= >= is as`                                            |
| 相等               | `==  !=`                                                     |
| 逻辑“与”           | `&`                                                          |
| 逻辑 XOR           | `^`                                                          |
| 逻辑 OR            | `|`                                                          |
| 逻辑 AND           | `&&`                                                         |
| 条件 OR            | \`\|\|`                                                      |
| null 合并          | `??`                                                         |
| 条件               | `?:`                                                         |
| 赋值和lambda表达式 | `= *= /= %= += -= <<= >>= &= ^= \|= \>`                      |

- 操作符（Operator）也译为“运算符”

- 操作符是用来操作数据的，被操作符操作的数据称为操作数（Operand）

  > 注：表中越上的的操作符优先级越高
